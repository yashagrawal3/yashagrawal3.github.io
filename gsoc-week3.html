GSoC Blog - Week 3

This is my fourth GSoC blog which tells work done this week.

Work Done This Week:

This week I worked on following activities:

  • Pyeyes Activity
  • Ajedrez Activity
  • Moon Activity
  • Turtle Confusion

Pyeyes Activity

All the remaining work in this acitivity was done and port to GTK+ 3 was completed.

Ajedrez Activity

All Files of this activity was ported to GTK+ 3 in this week. Since this activity had some bugs initially in the previous version, these bugs were still present after the port. Later after discussion between James and Tony, James metioned that I should deffer work on this activity.

Moon Activity

Most of the work on this activity was done by Ibiam already, some part of this activity is not functional. Even after hours of debugging, few bugs were still left. After discussion with Ibiam, we decided to deffer this activity to later until Ibiam is done with his tests.

Turtle Confusion

This is a big activity file based on Turtle Art. It was made to make starting with Turtle Art easier for students. Initially I started working with porting a lot of files to GTK+ 3, but later after suggestion from James, I decided to use git format-patch and git am to bring all changes between the activities and resolve conflicts, which itself prooved to be a time taking task.
